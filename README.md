🌌 QuickPoll

QuickPoll is a lightweight web app that allows users to create and share polls instantly without sign-up.
It focuses on simplicity, visual polish, and smooth interaction while remaining completely client-side.

Create a poll, share the link, and let others vote instantly.

🚀 Live Demo

🔗 https://deadvedant.github.io/Quick-Poll/

✨ Features
🗳 Instant Poll Creation

Create polls with multiple options and share them immediately using a generated link.

📊 Animated Results

Poll results are displayed with smooth animated progress bars that visually grow to represent vote percentages.

🎨 Modern Glass UI

The interface uses glassmorphism, gradients, and smooth animations to provide a modern experience.

🌌 Interactive Background

QuickPoll V2 features a ParticleTS powered background with glowing interactive particles that react to mouse movement.

📱 Mobile Friendly

The interface adapts well to mobile screens with improved spacing and optimized effects.

🔗 Easy Sharing

Users can share polls instantly via:

Copy link

WhatsApp sharing

👥 Voter Tracking

Poll creators can view the list of voters and their selected options.

⚡ Custom Modal System

HTML5

CSS3

🧠 How It Works

QuickPoll uses localStorage to store poll data locally in the browser.

Each poll generates a unique URL using a hash-based ID, allowing users to share and access specific polls.

Example:

https://deadvedant.github.io/Quick-Poll/#poll-123456

This keeps the app fully client-side and serverless.
